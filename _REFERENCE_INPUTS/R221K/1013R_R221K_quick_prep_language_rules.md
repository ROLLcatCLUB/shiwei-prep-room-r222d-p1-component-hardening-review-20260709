# 1013R R221K Quick Prep Language Rules

```text
template_mode=quick_prep
density=low
runtime_enabled=false
```

Quick prep mode keeps only:

```text
one opening cue
one key question
one task instruction
one evidence reminder
```

It must not:

```text
generate long question chains
pretend to be a public lesson transcript
hide teacher confirmation points
invent source-specific content
```

Quick prep language is a compact prompt set, not a full lesson script.

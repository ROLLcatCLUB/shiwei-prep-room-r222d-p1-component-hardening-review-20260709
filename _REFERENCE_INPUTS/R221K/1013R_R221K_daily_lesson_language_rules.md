# 1013R R221K Daily Lesson Language Rules

```text
template_mode=daily_lesson
density=medium_low
runtime_enabled=false
```

Daily lesson mode keeps:

```text
concise opening
core question
student task instruction
one or two useful follow-ups
assessment/evidence reminder
```

It cuts:

```text
full public-lesson transcript language
long transitions
performance-style teacher talk
over-detailed feedback variants
```

Validation:

```text
daily language must be usable in class without reading a long script.
```

## Daily Prep Rubric

| Dimension | A- line | Fail line |
| --- | --- | --- |
| Tomorrow usability | Teacher can organize the class without rewriting the whole lesson | Only a topic outline |
| Time rhythm | Has approximate time blocks and flow | Activity order exists but no rhythm |
| Classroom organization | States individual/pair/group/class routines | Only says students discuss or create |
| Learning support | Has minimal scaffold, worksheet or screen prompt | Only mentions support abstractly |
| Art-specific guard | Keeps lesson type flavor | Collapses into generic class |
| Assessment evidence | Observable evidence and micro-rubric | Generic participation or praise |
| Teacher language | Natural, grade-fit, action-oriented | Thesis tone, slogan tone, AI tone |
| Source boundary | Marks system reasoning, assumptions, gaps or confirmation | Pretends all details are source facts |

## Quick Prep Rubric

| Dimension | A- line | Fail line |
| --- | --- | --- |
| Readability | Teacher can read it within 3 minutes | Too long or still a detailed plan |
| Minimum output | Goal, steps, material, prompt, evidence, gaps | Only goal + process |
| Lesson-type guard | Keeps the core guard visible | Collapses into generic template |
| Evidence bottom line | Names visible product/process evidence | No observable evidence |
| Teacher language | One or two usable prompts | Empty slogans |
| Lightweight boundary | Does not become daily prep | Overloaded with details |

## Stage Judgment

R222C targets:

```text
daily_prep: from B+ usable to A- teachable
quick_prep: keep A- lightweight quality
```

It does not target public lesson detail or formal UI implementation.

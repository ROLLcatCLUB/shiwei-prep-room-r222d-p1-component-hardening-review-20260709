## 美术大家庭

**目标**：认识几类美术大家庭成员；能说出一种美术与生活的联系。

**3-5 个环节**
1. 找：从教室或校园图片里找美术痕迹。
2. 分：把例子放进美术大家庭。
3. 连：画一张小关系图。
4. 说：介绍一个成员和它的作用。

**材料**：门类图卡、校园图片、关系图纸。

**关键提示语**：美术不只在书上，也在校园和生活里；请说清它在哪里、有什么用。

**最小产出 / 证据底线**：至少留下一个生活例子和一条关系图连接，不能只背门类名称。

**系统假设**：本课仍以欣赏评述和生活连接为主要路径。

**资料缺口**：未接入真实教材图片和本班校园照片。

**高风险确认**：最终成果采用关系图还是成员介绍会？

## 有趣的纸印

**目标**：比较纸材带来的印痕差异；尝试一种印法完成小幅纸印。

**3-5 个环节**
1. 摸：比较纸材厚薄、软硬和肌理。
2. 试：用一种方法试印并观察印痕。
3. 做：选择纸材完成小幅纸印。
4. 说：说出纸材、印法和效果。
5. 清：按小组分工整理工具。

**材料**：不同纸材、拓印或滚印工具、颜料或油墨、试印纸、清洁材料。

**关键提示语**：先看印痕变化，再评价作品；圈出最清楚或最有层次的一次；工具用完要分工整理。

**最小产出 / 证据底线**：至少保留一条试印记录，能说出纸材、印法或印痕效果。

**系统假设**：课堂允许基础试印和工具清理。

**资料缺口**：未确认学校可用油墨、滚筒或替代材料。

**高风险确认**：先自由试纸材，还是先示范一种关键印法？

## 有趣的文字和图画

**目标**：发现文字和图画的关系；设计一张图文字卡或一个小故事片段。

**3-5 个环节**
1. 猜：看象形字和图画，猜它表示什么。
2. 比：比较图像到文字的变化。
3. 看：找线条、形状、方向和组合关系。
4. 做：设计一张图文字卡或一句图文故事。

**材料**：象形字例图、字卡纸、彩笔。

**关键提示语**：我们看的是文字怎样像图画，不是只认字；你的图画要帮助别人理解字义。

**最小产出 / 证据底线**：至少完成一张字卡，并能说明文字和图画的关系。

**系统假设**：可使用象形字图片作为观察起点。

**资料缺口**：未接入本课具体教材图例。

**高风险确认**：先研究象形字，还是先玩字卡游戏进入？

## 会说话的手

**目标**：观察手的动作和形态；用手势、手印或小剧场表达角色和情绪。

**3-5 个环节**
1. 看：观察手的张开、弯曲、握紧和方向。
2. 记：画或印一个有特点的手势。
3. 变：把手势变成角色或符号。
4. 演：小组展示一个短小片段。

**材料**：手势图片、手印材料、剧本小卡。

**关键提示语**：先看手会说什么，再进入表演；请让大家看出你的手势表达了什么。

**最小产出 / 证据底线**：至少留下一个手势观察记录或手印角色，不只做口头表演。

**系统假设**：课堂支持小组短展示。

**资料缺口**：未确认是否使用手影、手印或绘画为主要媒介。

**高风险确认**：先观察手的特点，还是先从手影故事进入？
